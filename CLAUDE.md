# e pot of piphany

A stoner-themed game app deployed on **Vercel** using **Next.js**.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Deployment:** Vercel
- **Styling:** Tailwind CSS
- **Language:** TypeScript

## Game Concepts (Pick One!)

### 1. Trivia / Quiz — "Deep or Dumb?"
- Rate random epiphanies as profound or nonsense
- Stoner science trivia, weird facts, philosophy questions

### 2. Word Game
- Connect seemingly unrelated words/concepts to form an "epiphany"
- Mad Libs-style story generator with community voting

### 3. Party Game (Pass-the-Phone)
- Pass-the-phone style prompts for groups
- "Convince your friends" debate game on absurd topics
- Drawing/guessing game with trippy prompts

### 4. Idle / Clicker Game
- Grow a virtual garden, unlock new strains
- Collect epiphanies to level up your "third eye"

### 5. Puzzle Game
- Trippy visual puzzles that shift and morph
- Pattern matching with psychedelic aesthetics
- "Solve it before the munchies hit" timed challenges

### 6. Adventure / Story Game
- Choose-your-own-adventure through bizarre philosophical scenarios
- Text-based exploration with branching epiphanies

## Design Principles

- **Mobile-first** — this is a couch app
- **Simple UI** — large buttons, high contrast, minimal navigation
- **Trippy aesthetics** — psychedelic colors, smooth animations
- **Voice input** — for capturing thoughts easily

## Commands

```bash
npm run dev    # Start dev server
npm run build  # Production build
npm run lint   # Run linter
```
